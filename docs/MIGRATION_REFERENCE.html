  • Different widget systems: Jira uses gadgets, OP uses built-in widgets

  • Third-party gadgets: Marketplace gadgets have no OP equivalent

alternative

  3. Create Saved Queries (My Page ‚Üí Add widget)
  4. Use Work Package tables with filters and grouping
  5. Consider external BI tools (Metabase, Grafana) connected to OP API
